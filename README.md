# Use-irreader-on-linux

在 linux 上使用 irreader 的方法

1. 安装 CrossOver
2. 在 CrossOver 里安装 irreader, 软件选 Unlist(show all ,选最下面的), installer 选 exe, Bottle 选 win7 x64
3. 会提示缺几个组件, 按照说明去网站安装.
4. 在 2 的 Bottle 里安装 yahei 字体, English 改成 Chinese 安装中文字体. <https://www.crossoverchina.com/faq/rq-azzt.html>
5. 打开软件，左上角齿轮，填入对应的地址，enjoy!

---

Q: 怎么安装 CrossOver?

A: [Debian](https://www.crossoverchina.com/xiazai.html), [ArchLinux](https://wiki.archlinux.org/title/CrossOver) 

Q: win7 x86_64?

A: 软件需要 x86_64 环境, win10 安装后每次启动会弹 3 个报错, 不影响使用, 影响心情. wine 对老版本支持较好

Q: 为何安装字体?

A: 解决缺字、 字体变方块、 字体 fallback 等问题

Q: 不用更高版本?

A: 更高版[限制 10 个订阅](https://fatecore.com/threads/irreader/67/buy-irreader/)

Q: CrossOver 有时间限制

A: 网上查方法
